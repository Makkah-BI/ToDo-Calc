# ToDo-Calc
#To-Do List App – Core concepts used
CRUD operations (Create, Read, Update, Delete) – the backbone of most web apps

localStorage – persist tasks across page reloads

Event delegation – handling clicks on dynamically added delete/edit buttons

Form submission – prevent page refresh, capture input values

Array methods – map, filter, find to update/manage tasks

 #Calculator – Sharpens different muscles
Math logic – operator precedence (does 2+3*4 = 14 or 20?) 

Keyboard support – listen to keydown events, map number keys, Enter, Backspace, Escape

UI state – handle chained operations (5 + 3 = then * 2 = should continue)

Edge cases – division by zero, decimal point limits, very long numbers overflow
A little different from the normal calculators.
